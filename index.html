Portfolio

Podívej se na moje projekty, které jsou seřazeny od nejnovější po nejstarší. Můžeš si všimnout, že s každým projektem jsem se posunula na další úroveň. Před novým projektem si vytvořím cíle, kterých chci dosáhnout a učit se novým věcem.

Dark Portfolio

HTML, CSS

Nejnovější projekt "Dark Portfolio" je vlastní práce, kde vytvářím svou stránku pomocí nabytích vědomostí z předchozích projektů. Pracuji i s internetem a nechávám se inspirovat, aby web působil přirozeně.

 

Trillo

HTML, CSS

Projekt Trillo byl vytvořen společně s "Jonášem" (Jonas Schmedtmann). Projekt je vytvořen za pomocí flexboxu. Nicméně "nefunkční", protože to nebylo cílem projektu. Cílem bylo naučit se flexbox a responzivitu.

 

Natours

HTML, CSS

Projekt Natours byl vytvořen společně s "Jonášem" (Jonas Schmedtmann). Projekt využívá pokročilých CSS vlastností. Cílem bylo prohloubit mé znalosti CSS a naučit se základy responzivity.

 

TinCat

HTML, CSS

Projekt TinCat byl vytvořen společně s Dr. Angelou Yu, která vytvářela "TinDog". V tomto projektu jsem se začínala učit pořádně CSS. TinCat rozhodně není dokonalý projekt, ale dokazuje, že se neustále posouvám a vzdělávám. Jakožto milovnice koček byla pro mě výzva vyměnit všechny fotografie na kočičí fotky. Samozřejmě i přepsat texty. Cílem bylo naučit se základy CSS.

 

Code Word

HTML, CSS

Po zvládnutí základů CSS jsem si chtěla procvičit nabyté vědomosti a vytvořit vlastní projekt "CodeWord". Byl využit BOOTSTRAP.

 

AmateursCoffe

HTML, CSS

Po zvládnutí základů CSS jsem si chtěla procvičit nabyté vědomosti a vytvořit vlastní projekt "AmateurCoffee". Byl využit BOOTSTRAP. I přes fakt, že AmateurCoffee vznikl dříve než "CodeWord", mnou vytvořený design se mi líbí více než u CodeWord.
